# Telegram-API-fix

## Фикс сайтов, такие как: 
- web.telegram.org
- core.telegram.org
- api.telegram.org

## Установка:
- запуск telegram_unblock.bat от имени администратора

## Как всё вернуть, как было?
- запуск telegram_restore.bat от имени администратора
